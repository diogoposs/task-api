# Task API

REST API for task management built with Spring Boot.

## 🚀 Technologies
- Java 17
- Spring Boot
- Maven
- H2 Database
- JUnit
- Rest Assured (API Tests)
- Cucumber (BDD)

## 📌 Features
- Create Task (POST)
- Get Task (GET)
- Update Task (PUT)
- Delete Task (DELETE)

## ▶️ How to Run
```bash
mvn spring-boot:run
